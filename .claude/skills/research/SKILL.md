# /research - 統合検索スキル

海外のYouTube・TikTok・X(Twitter)を横断検索し、結果を分類して表示する。

## 使い方
```
/research <検索キーワード>
```

## 実行手順

1. **検索の実行**: WebSearchツールを使って以下の検索を並列実行する
   - `<キーワード> site:youtube.com`
   - `<キーワード> site:tiktok.com`
   - `<キーワード> site:x.com OR site:twitter.com`
   - 必要に応じて英語キーワードで検索（日本語キーワードが渡された場合も英語に翻訳して検索）

2. **取捨選択**: CLAUDE.mdの基準に従い、低品質・スパム的なものを除外

3. **分類**: 各結果を以下の3カテゴリに分類
   - **公式・アップデート**: 公式アカウント・ブログからのリリース情報
   - **ユーザー発見・活用法**: ユーザー独自のTips・ワークフロー
   - **トレンド・バズ**: バズっている投稿・急上昇中の話題

4. **出力**: カテゴリごとにテーブル形式で表示（通し番号付き）
   ```
   ## 検索結果: "<キーワード>"

   ### 公式・アップデート
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|

   ### ユーザー発見・活用法
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|

   ### トレンド・バズ
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|
   ```

5. **保存**: 結果を `results/latest.md` に書き込む（以前の内容を上書き）
   - ファイルには検索キーワード、検索日時、全結果（番号・タイトル・URL・カテゴリ・ソース）を含める

## 注意事項
- 日本語で出力すること
- 「できること・学べること」列は必須。具体的な価値を1行で記述する
- 各カテゴリに該当がない場合は「該当なし」と表示
- 番号は全カテゴリ通しで振る（/detail で参照するため）
- 検索キーワードが日本語の場合、英語に翻訳して検索も行う
