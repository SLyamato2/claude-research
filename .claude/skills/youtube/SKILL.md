# /youtube - YouTube特化検索スキル

YouTube上の動画を検索し、結果を分類して表示する。

## 使い方
```
/youtube <検索キーワード>
```

## 実行手順

1. **検索の実行**: WebSearchツールで以下を検索
   - `<英語キーワード> site:youtube.com`（3カテゴリすべて採用）
   - `<日本語キーワード> site:youtube.com`（トレンド・バズのみ採用）
   - 日本語キーワードが渡された場合は英語に翻訳して英語検索も行う
   - 英語キーワードが渡された場合は日本語に翻訳して日本語検索も行う

2. **取捨選択**: CLAUDE.mdの基準に従い、低品質・スパム的なものを除外

3. **分類**: 各結果を3カテゴリに分類
   - **公式・アップデート**: 公式チャンネルからのリリース解説動画
   - **ユーザー発見・活用法**: ユーザーによるチュートリアル・Tips動画
   - **トレンド・バズ**: 再生数が急伸している動画・話題の動画

4. **出力**: カテゴリごとにテーブル形式で表示（通し番号付き）
   ```
   ## YouTube検索結果: "<キーワード>"

   ### 公式・アップデート
   | # | タイトル | できること・学べること | 日付 | URL | ソース |
   |---|---------|---------------------|------|-----|--------|

   ### ユーザー発見・活用法
   | # | タイトル | できること・学べること | 日付 | URL | ソース |
   |---|---------|---------------------|------|-----|--------|

   ### トレンド・バズ
   | # | タイトル | できること・学べること | 日付 | URL | ソース |
   |---|---------|---------------------|------|-----|--------|
   ```

## 注意事項
- 日本語で出力すること
- ソース列はすべて「YouTube」
- 「できること・学べること」列は必須
- 番号は全カテゴリ通しで振る
- **結果のmdファイル保存は行わない**（画面表示のみ）
- **日本語ソースはトレンド・バズのみ採用**
