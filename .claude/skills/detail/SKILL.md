# /detail - 番号指定で詳細解説スキル

検索結果の番号を指定して、その情報の詳細解説を表示する。

## 使い方
```
/detail <番号>
```

## 実行手順

1. **結果の読み込み**: `results/latest.md` を読み込み、指定された番号の項目を特定する

2. **情報取得**: WebFetchツールで該当URLの内容を取得する

3. **詳細解説の出力**: 以下のフォーマットで日本語で出力する
   ```
   ## #<番号> の詳細解説

   **タイトル:** <タイトル>
   **URL:** <URL>
   **ソース:** <YouTube/TikTok/X>
   **カテゴリ:** <公式・アップデート/ユーザー発見・活用法/トレンド・バズ>

   ### 内容の詳細
   （WebFetchで取得した情報を元に、内容を詳しく日本語で解説）

   ### できること・実践方法
   （実際のコマンド例、設定例、ワークフローなど具体的なアクション）

   ### ポイント・注意点
   （重要な注意点、前提条件、制限事項など）
   ```

## 注意事項
- 日本語で出力すること
- `results/latest.md` が存在しない、または番号が見つからない場合はエラーメッセージを表示
- WebFetchで取得できない場合は、検索結果の情報から可能な限り解説する
- 具体例やコマンド例は可能な限り含める
