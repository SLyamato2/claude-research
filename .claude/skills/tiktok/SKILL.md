# /tiktok - TikTok特化検索スキル

TikTok上の動画を検索し、結果を分類して表示する。

## 使い方
```
/tiktok <検索キーワード>
```

## 実行手順

1. **検索の実行**: WebSearchツールで以下を検索
   - `<キーワード> site:tiktok.com`
   - 日本語キーワードの場合は英語に翻訳して追加検索

2. **取捨選択**: CLAUDE.mdの基準に従い、低品質・スパム的なものを除外

3. **分類**: 各結果を3カテゴリに分類
   - **公式・アップデート**: 公式アカウントからの発表動画
   - **ユーザー発見・活用法**: ユーザーによるTips・活用法の短尺動画
   - **トレンド・バズ**: バズっている動画・急上昇中のコンテンツ

4. **出力**: カテゴリごとにテーブル形式で表示（通し番号付き）
   ```
   ## TikTok検索結果: "<キーワード>"

   ### 公式・アップデート
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|

   ### ユーザー発見・活用法
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|

   ### トレンド・バズ
   | # | タイトル | できること・学べること | URL | ソース |
   |---|---------|---------------------|-----|--------|
   ```

5. **保存**: 結果を `results/latest.md` に書き込む（以前の内容を上書き）

## 注意事項
- 日本語で出力すること
- ソース列はすべて「TikTok」
- 「できること・学べること」列は必須
- 番号は全カテゴリ通しで振る
