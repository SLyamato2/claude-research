# 情報リサーチプロジェクト

このディレクトリは、海外および日本のYouTube・X(Twitter)・技術記事サイトから最新情報を取得し、分類・整理するためのリサーチ環境です。

## 対象ユーザー
- **主な読者はWeb開発者**（フロントエンド・バックエンド問わず）
- `/detail` の利用シーン解説は、Web開発者の視点を軸に記述する
- ただし、業務効率化・タスク管理・AI活用など開発以外の情報も、Web開発者の日常に役立つものは除外しない

## 応答言語
- **常に日本語で応答すること**

## 情報の分類ルール

検索結果は以下の3カテゴリに自動分類する：

| カテゴリ | 説明 | 例 |
|---------|------|-----|
| **公式・アップデート** | 公式アカウント・公式ブログからのリリース情報、機能追加、変更点 | 公式の新機能発表、公式チャンネルの解説動画 |
| **ユーザー発見・活用法** | ユーザーが見つけた新しい使い方、Tips、ワークフロー | 「こうやったら10倍速くなった」系の投稿、独自の活用術 |
| **トレンド・バズ** | バズっている投稿、急上昇中の話題、議論を呼んでいるもの | いいね/RT数が多い投稿、再生数が急伸している動画 |

## 検索対象プラットフォーム
- **YouTube** - 対象
- **X(Twitter)** - 対象
- **Dev.to** - 対象（海外技術記事）
- **Hashnode** - 対象（海外技術記事）
- **Hacker News** - 対象（海外テック系ニュース・議論）
- **Zenn** - 対象（日本語技術記事）
- **note** - 対象（日本語技術記事）
- **TikTok** - **除外**（動画コンテンツの詳細取得が困難なため）

## 取捨選択の基準

### 含めるもの
- 実用性が高く、具体的なアクションにつながる情報
- 公式ソースからの一次情報
- 独自の知見・発見がある投稿
- エンゲージメントが高く話題になっているもの

### 除外するもの
- スパム的・クリックベイト的なコンテンツ
- 内容が薄い・既知の情報の焼き直し
- 宣伝目的のみの投稿
- 信頼性の低いソース

## 検索ソースの優先度

### 海外（英語）ソース
- 3カテゴリすべて（公式・アップデート / ユーザー発見・活用法 / トレンド・バズ）を採用

### 日本語ソース
- **トレンド・バズのみ採用**（本当にバズっているものだけに絞る）
- 日本語の「公式・アップデート」「ユーザー発見・活用法」は基本的に除外

## 出力フォーマット規約

### 検索結果の表形式
各カテゴリごとにテーブルで表示。**「できること・学べること」列は必須**。

```
| # | タイトル | できること・学べること | 日付 | URL | ソース |
|---|---------|---------------------|------|-----|--------|
```

- 番号は全カテゴリ通しで振る（`/detail`で参照するため）
- **日付**は投稿日・公開日・更新日を `YYYY-MM-DD` 形式で記載（検索結果から判別できる範囲で。不明な場合は「不明」と記載）
- ソースは YouTube / X / Dev.to / Hashnode / Hacker News / Zenn / note のいずれかを明記
- 「できること・学べること」は、その情報から読者が得られる具体的な価値を1行で記述

### 結果の保存
- 検索結果のmdファイル保存は**行わない**（画面表示のみ）
